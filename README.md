# Kurs z analizy oraz wizualizacji w pythonie

Zespół: Klaudia Malecka, Cyprian Księżopolski, Jakub Klimek, Patryk Szweda

## Cele projektu

1. Nauka przetwarzania, analizowania i wizualizacji danych
2. Wzbogacenie umiejętności pracy grupowej za pomocą githuba
3. Poszerzanie umiejętności w posługiwaniu się językiem python w celach najpierw preprocesingu danych, a następnie ich wizualizacji

## Co robi nasz projekt?

W naszym projekcie skupiamy się na zbiorze danych lotów z 2013 roku.  
Dane są czyszczone, uporzadkowywane i normalizowane tak, aby były gotowe do późniejszej wizualizacji oraz analizy.
